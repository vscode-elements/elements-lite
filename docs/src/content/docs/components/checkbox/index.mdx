# Indeterminate checkbox

Indeterminated state is applied on checkbox. [MDN Reference](https://developer.mozilla.org/en-US/docs/Web/API/HTMLInputElement/indeterminate)

<VscodeDemo>
  <IndeterminateExample />
</VscodeDemo>
